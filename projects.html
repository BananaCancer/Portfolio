February 2024 - April 2024 (3 months)

Academic project, UQAC, Chicoutimi, Québec, Canada

Project description
In this project we were a team of two that did a literature review on the use of reinforcement learning to train agents in video games. We also used some of these articles to recreate the experiments that were done in order to compare them. In the end, it allowed us to create/reuse games using python and Unreal Engine and to train agents using algorithms such as Q-Learn, Deep Reinforcement Learning and the PPO algorithm.

Soft skills
Perseverance Curiosity

  1. Writing a literature review

    After choosing our topic, we did a deep research on the topic of reinforcement learning for video game agents and synthetised it all in our review. It was a very interesting process that improved our skills in vulgarization, research and overall reinforcement learning knowledge.

  2. Reimplementing research papers process

    One of the most interesting part of our project was to recreate some of the papers results. We studied 3 papers thoroughly and used the same processes (but adapted to different games) to obtain similar results. Though the task was consequent due to the variety of used technologies, overcoming the various implementation differences to obtain interesting results makes this project one of the most interesting projects I've been a part of.

  3. Comparing results and critical thinking

    After recreating the papers' results, we also compared the results of the various algorithms and games we used. This allowed us to conclude the project by doing some critical thinking on what we produced vs what we expected.

Data science project member

November 2023 - December 2023 (2 months)

Academic project, UQAC, Chicoutimi, Québec, Canada

Project description
Academic project where we used machine learning to understand and help improve the sales of a fake company. The data used was found on Kaggle. We used clustering and association rules to find some relations between certain clients and the products they bought. Thanks to these methods, we successfully managed to find solutions to improve the sales of the various products of the company.
Technical skills
Soft skills
Data analysis Communication
Teamwork Presentation skills
Tasks

  1. Client clusters creation

    I tested multiple clustering algorithms to regroup the clients depending on their profile (income/recency/expenses). I tested multiple types of algorithms (GMM, KMeans) and judged their quality mainly using their silhouette.

  2. Client profil association

    Using association rules, we associated different client features to help the analysis of the data. We then analyzed the results to give some guidelines to the company so that they can improve their sales for some products.

  3. Carefully preparing data

    We didn't stop at doing a simple ML workflow. In order to have the best results possible, we carefully prepared the data to chose the wanted features. We also created new features out of the existing ones to make the dataset simpler for the clustering phase.
